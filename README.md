# linux-hardening-lab
Linux infrastructure hardening and administration lab using RHEL 10 and Ubuntu Server in VMware Workstation Pro.

## Overview

This project documents a Linux infrastructure and security hardening lab built using VMware Workstation Pro, Red Hat Enterprise Linux 10, and Ubuntu Server 24.04 LTS.

The purpose of the lab is to strengthen hands-on Linux administration, security monitoring, automation, and infrastructure engineering skills in a multi-VM environment.

---

## Lab Environment

### Virtualization Platform
- VMware Workstation Pro

### Operating Systems
- RHEL 10 Minimal
- Ubuntu Server 24.04 LTS

### VM Architecture

| Hostname | Purpose |
|---|---|
| rhel-admin01 | Linux administration and hardening |
| ubuntu-docker01 | Docker and container services |
| secmon01 | Wazuh monitoring and security platform |
